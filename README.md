# DevelopersHub_Advanced_Task5_BusinessDashboard
Interactive business dashboard built with Streamlit to analyze sales, profit, and customer performance using the Superstore dataset.

## 📌 Objective
The objective of this project is to develop an interactive business dashboard to analyze sales, profit, and customer performance using the Global Superstore dataset.

---

## 📊 Dataset
The dataset used is the Global Superstore dataset, which includes:

- Sales and Profit data  
- Customer information  
- Product categories and sub-categories  
- Regional sales distribution  

---

## ⚙️ Features

### 🔎 Interactive Filters
- Region
- Category
- Sub-Category

### 📊 Key Performance Indicators (KPIs)
- Total Sales  
- Total Profit  
- Total Orders  

### 📈 Visualizations
- Sales by Category  
- Profit by Region  

### 🏆 Top Customers
- Top 5 customers based on total sales  

---

## 🛠️ Tools & Technologies

- Python  
- Streamlit  
- Pandas  
- Plotly  

---

## 🚀 How to Run the Project
1. Clone the repository
2. Install required libraries:
3. pip install pandas streamlit openpyxl plotly
4. Run the app:
5. streamlit run app.py
